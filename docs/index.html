Project Description

A modern day problem that many people face is having to endure a party host's terrible taste in music. Troubadour seeks to give the world a comprehensive solution to this by generating an optimized playlist based off the preferences of party attendees. Powered by Spotify's API, we can instantly generate a playlist that everyone will enjoy.

Project Scope

Troubadour consists of an Android application, a backend server, and a database server. We will provide an interface for entering music preferences as well as an interface for generating a party playlist. We will implement an algorithm to aggregate musicp references for playlist generation.
